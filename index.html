Business site

Strona biznesowa

A simple website mockup for a fake legal company. Created to practice using the React library, it features a clean design with a vertical navbar and a custom logo. React-router is used to redirect the user, animations added using react-animate-on-scroll. Created with HTML, CSS and React (hooks).

Prosta strona-wizytówka dla fikcyjnej firmy prawniczej. Stworzona w celu przećwiczenia korzystania z biblioteki React, charakteryzuje się prostym i przejrzystym wyglądem z wbudowanym pionowym paskiem nawigacyjnym oraz specjalnie stworzonym logotypem. Do nawigacji wykorzystany został React-router, do animacji react-animate-on-scroll. Napisana w HTML, CSS i React (hooks).

World clocks

Zegary świata

An app which lets you check the time in any timezone in the world and display up to 6 clocks from different timezones. My first attempt at creating a complete React project. The app fetches data from a free API and displays working clocks. Created with HTML, CSS and React (hooks).

Aplikacja pozwalająca na sprawdzenie godziny w dowolnej strefie czasowej na świecie i wyświetlenie do 6 zegarów z różnych stref czasowych na raz. Moja pierwsza próba stworzenia kompletnego projektu przy użyciu Reacta. Aplikacja pobiera dane dotyczące stref czasowych z darmowego API i wyświetla działające zegary analogowe. Napisana w HTML, CSS i React (hooks).

Live Github

Medicine tracker

Katalog leków

This app lets you easily keep track of your medicines. It automatically labels expired / soon to expire items, lets the user sort data by name, expiration date and type and allows for a keyword search in the database. Data is stored externally in a JSON file and is not editable. Created with HTML, CSS and Javascript.

Aplikacja do katalogowania leków. Sama zaznacza kolorami przeterminowane lekarstwa oraz te, których termin ważności niedługo się kończy. Pozwala na sortowanie listy po nazwie, dacie ważności oraz opisie, a także umożliwia przeszukiwanie listy. Dane przechywane są w zewnętrznym pliku JSON. Dodawanie i usuwanie leków działa lokalnie, lecz nie modyfikuje bazy danych. Napisana w HTML, CSS i Javascript.

Live Github

Memory game

Gra pamięciowa

My take on the classic tiles memory game. This one has a timer built in, so you have to be fast! Three levels of difficulty and three different tilesets. Music and sound. Victory screen with points and a star rating. Game made with free assets found online. Created with HTML, CSS and Javascript

Moja wersja klasycznej gry pamięciowej. Posiada wbudowany licznik czasu, więc musisz być szybki/a! 3 poziomy trudności, 3 różne zestawy obrazków. Dźwięk i muzyka. Ekran końcowy z ilością punktów i uzyskanymi gwiazdkami. Gra wykonana z pełni darmowych zasobów znalezionych w sieci. Napisana w HTML, CSS i Javascript.

Writer portfolio

Portfolio pisarza

My first serious project - a portfolio site for an fantasy author. Simple, vanilla design. It features a custom news sorting system and a working PHP contact form. As this was my first project, every part of it was somewhat of a challenge, but at the same time a great learning opportunity. Made with HTML, CSS and Javascript + a few lines of PHP.

Mój pierwszy poważny projekt - strona typu portfolio dla pisarza fantasy. Stonowany, prosty wygląd. System sortowania newsów, działający formularz kontaktowy obsługiwany przez PHP. Ze względu na to, że był to mój pierwszy projekt, każdy jego element stanowił pewne wyzwanie, ale i też okazję do nauki i szlifowania umiejętności. Napisany w HTML, CSS i Javascript + parę linijek PHP.

About meO mnie

Hi, my name is Jakub Jaskuła and I'm a front-end developer from Łask near Łódź (Poland). I love the creative process and always strive to deliver clean, intuitive, dynamic user interfaces. I design and code with simplicity in mind, trying to keep the code readable and organized. I like to learn new things and I think I'm a pretty quick study too.

I speak Polish (native), English (C2) and German (basic).

When I'm not working I like to spend time with my family and relax. I also enjoy hiking, cycling and playing video games. I've never said "no" to a good book either!

Hej, nazywam się Jakub Jaskuła i jestem front-end developerem z Łasku koło Łodzi. Uwielbiam tworzyć nowe rzeczy, a w moich projektach zawsze dążę do uzyskania przejrzystego, intuicyjnego a zarazem dynamicznego interfesju użytkownika. Projektę i programuję z myślą o prostocie wykonania, tak by mój kod był łatwy do odczytania i dobrze zorganizowany. Lubię się uczyć i myślę, że szybko przyswajam wiedzę.

Mówię po polsku, angielsku (C2) i niemiecku (podstawy)

Kiedy nie pracuję lubię spędzać czas z moją rodziną. Poza tym chętnie spaceruję, jeżdżę na rowerze i gram w gry. Dobrą książką też nie pogardzę!
