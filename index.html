Business site

Strona biznesowa

A simple business website mockup made with React and create-react-app. Created to practice using the React library, it features a clean design with a vertical navbar and a custom logo. React-router is used to redirect the user, animations added using react-animate-on-scroll

Prosta strona-wizytówka dla fikcyjnej firmy prawniczej, wykonana przy użyciu Reacta i create-react-app. Stworzona w celu przećwiczenia korzystania z biblioteki React, charakteryzuje się prostym i estetycznym wykonaniem z wbudowanym pionowym paskiem nawigacyjnym i specjalnie stworzonym logotypem. Do nawigacji wykorzystany został React-router, do animacji react-animate-on-scroll.

World clocks

Światowe zegary

A React-based app which lets you check the time in any timezone in the world and display up to 6 clocks from different timezones. My first attempt at creating a React project. The app fetches data from a free API. The analog clock is inspired by a "JavaScript30" course challenge by Wes Bos, but I've added some features

Aplikacja napisana w React, pozwalająca na sprawdzenie godziny w dowolnej strefie czasowej na świecie i wyświetlenie do 6 zegarów z różnych stref czasowych na raz. Moja pierwsza próba stworzenia projektu przy użyciu biblioteki React. Aplikacja łączy się z darmowym API i pobiera dane dotyczące stref czasowych. Zegar analogowy inspirowany zadaniem z kursu Wes Bos'a 'JavaScript30' z moimi modyfikacjami.

Medicine box

System sortowania leków

Medicine box lets you keep track of your medicines, their indications for use and expiration date. It automatically labels expired / soon to expire items, lets the user sort data by name, expiration date and type and allows for a keyword search in the database. Data is stored externally in a JSON file and is not editable.

Aplikacja do katalogowania posiadanych lekarstw, ich zastosowania oraz daty ważności. Automatycznie zaznacza kolorami przeterminowane lekarstwa oraz te których termin ważności się niedługo kończy. Pozwala użytkownikowi na sortowanie listy po nazwie, dacie ważności oraz opisie, a także umożliwia przeszukiwanie listy. Dane przechywane są w zewnętrznym pliku JSON. Dodawanie i usuwanie leków działa lokalnie, lecz nie modyfikuje bazy danych.

Szybka pamięć

A puzzle game, one of my first projects. Working on it gave me the opportunity to practice using CSS Grid, arrays and array methods. One of the challenges here was making sure the game is responsive on different screen sizes. Game made with free assets found online

Pewną trudnością było dostosowanie gry do różnych rozmiarów ekranów. Gra wykonana z pełni darmowych zasóbów znalezionych w sieci.

Author portfolio

Portfolio pisarza

My first serious project - a portfolio site for an fantasy author. Simple, vanilla design. It features a custom news sorting and a working PHP contact form. As this was my first project, every part of it was somewhat of a challenge, but at the same time, a great learning opportunity.

Mój pierwszy projekt - strona typu portfolio dla pisarza fantasy. Stonowany, prosty wygląd. Brak zewnętrznych bibliotek czy wtyczek. Ze względu na to, że był to mój pierwszy projekt, każdy jego element stanowił pewne wyzwanie, ale i też okazję do nauki i szlifowania umiejętności. Wiele rzeczy robiłem po raz pierwszy - np. użycie PHP czy zakup i konfiguarcja domeny z hostingiem, a także zamieszczenie strony w sieci.
